# 🚀 AstroVerse - Space Explorer

AstroVerse is a modern and interactive space exploration website built using **HTML, CSS, and JavaScript**. The project provides a futuristic user experience where visitors can explore planets, space missions, galaxies, and fascinating facts about the universe.

---

## 🌌 Features

* 🚀 Modern Space-Themed UI
* ✨ Animated Star Background
* 🪐 Planet Showcase Section
* 🌍 Real Planet Images
* 📊 Statistics Cards
* 🎯 Featured Space Missions
* 🌌 Galaxy Gallery
* 💡 Random Space Facts Generator
* 🎨 Glassmorphism Design
* ⚡ Smooth Animations
* 🌠 Shooting Star Effects
* 💫 Cursor Glow Effect
* 📱 Fully Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 Sections

Explore planets with beautiful images and descriptions.

### Missions

Discover famous space missions including Apollo 11, Voyager 1, and Artemis.

### Galaxy Gallery

A collection of stunning cosmic visuals.

Random facts about the universe generated dynamically with JavaScript.

## ⚙️ Installation

1. Clone the repository

```bash

```

2. Open the project folder

```bash
cd astroverse
```

3. Run the project

```text
Open index.html in your browser
```
